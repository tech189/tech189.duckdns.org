Danny Bot

Overview

A Discord bot written in Python 3 for my server with features such as checking Minecraft server status, cryptocurrency price, weather, and others.

Features

Minecraft related:

  • !mc checks the server status
  • !list lists the current online players
  • !lastonline checks who was last to leave the server

Tools:

  • !crypto checks price and converts a few cryptos
  • !exam checks how long until exams
  • !weather checks the current weather

Fun:

  • !fry deep fries images you throw at it

Miscellaneous commands:

  • !pi checks the status of the pi
  • !speed runs a speedtest from the server
  • !about: display info about the bot
